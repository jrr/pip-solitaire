# Pip

This is a simple [PWA](https://en.wikipedia.org/wiki/Progressive_web_app) solitaire game. It started as an experiment with agentic development, and almost all of the code has been written by Claude.

The latest version is available at [jrr.github.io/pip-solitaire](https://jrr.github.io/pip-solitaire/). CI produces screenshot reports, visible at [jrr.github.io/pip-solitaire/screenshots/](https://jrr.github.io/pip-solitaire/screenshots/)
